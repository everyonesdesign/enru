# Enru-python
A better realization of [enru script](https://github.com/everyonesdesign/enru), this time based on python

Parses Ya.Dictionary service to get the translation

## Installation

```bash
$ chmod +x install.sh
$ ./install.sh
```

## Usage

```bash
$ enru 'query'
```
